经典部件：输入设备、输出设备、存储器、数据通路（运算器）、控制器

输入设备

为计算机提供信息的装置，如键盘、麦克风等

输出设备

将计算结果输出给用户或其他计算机装置，如显示器、扬声器等

显示器（PC）

液晶显示（Liquid Crystal Display, LCD）

背光层（位于分层结构最后面的一层由荧光物质组成的，可以发射光线）发出的光线在穿过第一层偏振过滤层之后进入包含成千上万水晶液滴的液晶层。液晶层中的水晶液滴都被包含在细小的单元格结构中，一个或多个单元格构成屏幕上的一个像素。当LCD中的电极产生电场时，液晶分子就会产生扭曲，从而将穿越其中的光线进行有规则的折射，然后经过第二层过滤层的过滤在屏幕上显示出来

  • 用液体聚合物薄层的带电或者不带电来传输或者阻止光线的传输
  • LCD 非光源，而是控制光的传输
  • 轻巧、低功耗

动态矩阵显示（active matrix display）

每一个像素都是由三个液晶单元格构成，其中每一个单元格前面都分别有红色，绿色，或蓝色的过滤器。这样，通过不同单元格的光线就可以在屏幕上显示出不同的颜色

  • 使用8位表示每个三原色（红、绿、蓝）
  • 像素：图像元素的最小单元，每个像素用24位表示，可以显示百万种不同的颜色
  • 使用晶体管控制单个像素上光线的传输，每个像素需要3个晶体管开关

计算机硬件采用光栅刷新缓冲区（也称帧缓冲区）来保存位图以支持图像。图像保存在帧缓冲区中，每个像素的二进制以刷新频率独处到显示设备

可参考:http://www.360doc.com/content/14/0317/12/1204156_361255697.shtml

触摸屏（后PC时代）

可采用多种方式实现，平板电脑使用电容感应实现，如果绝缘玻璃上覆盖一层透明的导体，人的手指接触到屏幕范围时，由于人是导体，将会使屏幕的电场发生变化，进而导致电容的变化，这种技术允许同事接触多个点。

中央处理单元（central processor unit，CPU）

寄存器（register）

寄存器数量比较少
寄存器中的数据更容易利用，一个算术运算指令可以完成读两个寄存器并对运算结果操作，而一条数据传输指令只能完成读或者写一个数
访问时间短
吞吐率搞
访问寄存器功耗小

数据通路（运算器 datapath）

处理器中执行算术操作的部分，负责完成算术运算

控制器（control）

处理器中根据程序指令指导数据通路、存储器和I/O设备按照程序的指令正确执行的部分

存储器

存储技术：

  • 静态随机访问存储器（Static Random Access Memory, SRAM）：集成电路形式的存储器，比DRAM更快集成度更低价格更贵
  • 动态随机访问存储器（Dynamic Random Access Memory, DRAM）：集成电路形式的存储器，可随机访问任何地址的内存。与串行访问（如磁带）不同的是，无论数据存储在什么位置，DRAM访问内存所需的时间基本相同

易失性存储器（volatile memory 也称主存储器（main memory））

仅在加电时保存数据，关闭电源，数据就会丢失

  • 保存运行中的程序

缓存（cache memory）

处理器内部，一般作为DRAM的缓冲，采用SRAM

内存

由DRAM芯片组成，程序运行时的存储空间，同时也存储程序运行时所需的数据

非易失性存储器（nonvolatile memory 也称二级存储器（main memory））

在掉电时仍可保持数据

  • 保存两次运行之间的程序和数据

磁盘（magnetic disk 也称硬盘（hard disk））

使用磁介质材料构成的以旋转盘片为基础的非易失性二级存储设备

闪存（flash memory）

在个人移动设备中使用闪存代替磁盘

  • 比DRAM慢但是更便宜
  • 比磁盘贵但是在体积、电容、可靠性和能耗方面都优于磁盘
  • 与硬盘和DRAM不同，闪存具有写100000~1000000次后老化或损坏的弱点。因此文件系统必须记录写操作的次数，并且具备避免存储器损坏的策略（例如：避免移动经常使用的数据）

处理器和存储器制造原理

计算机实现技术的发展

年代 计算机中采用的技术 相对性价比
1951 真空管 1
1965 晶体管 35
1975 集成电路 900
1995 超大规模集成电路 2400000
2013 甚大规模集成电路 25000000000

晶体管（transistor）：一种由电流控制的简单开关
集成电路（integrated circuit 简：IC）：由成千上万个晶体管组成的芯片
超大规模集成电路（very large-scale integrated circuit 简：VLSI）：由数十万到数百万晶体管组成的电路
摩尔定律：当价格不变时，集成电路（单芯片）上可容纳的元器件的数目，约每隔18-24个月便会增加一倍，性能也将提升一倍


集成电路的制作过程

原材料

导体：电阻率很小且易于传导电流的物质
绝缘体（电介质）：不善于传导电流的物质，电阻率极高
半导体：常温下导电性能介于导体（conductor）与绝缘体（insulator）之间的材料
硅：沙子中的一种物质

用特殊的化学方法对硅添加某些材料，改变其导电性，生成三种类型的材料

  • 良好的导电体（类似于细微的铜线或铝线）
  • 良好的绝缘体（类似于塑料或玻璃膜）
  • 可控的导电体或绝缘体（类似开关）

硅锭：一块由硅晶体组成的棒。直径大约在8~12英寸，长度约为12~24英寸
晶圆：厚度不超过0.1英寸的硅锭片
瑕疵：晶圆上微小的缺陷
芯片（chip）：从晶圆中切割出来的一个单独的矩形区域
成品率：合格芯片数占总芯片数的百分比

成品率=1/(1+(单位面积的瑕疵数芯片面积/2))^2
每晶圆的芯片数≈晶圆面积/芯片面积
每芯片的价格 = 每晶圆的价格/(每晶圆的芯片数成品率)

硬件设计原则

简单源于规整

操作数个数不可变，为硬件设计提供了很大的帮助

越小越好

寄存器个数限定为32，大量的寄存器可能会使时钟周期变长，因为电信号传输更远的距离必然花费更长的时间
