程序

程序是一组有序的指令序列及有关的数据

  • 程序是对计算任务的处理对象和处理规则的描述

二进制机器语言程序

机器语言：以二进制元形式表示的机器指令

汇编语言程序

汇编语言：以助记符形式表示的机器指令

高级语言程序

如C、C++、Java、Visual Basic 等可移植的语言，由一些单词和代数符号组成，可以由编译器转换为汇编语言

  • 高级语言的执行过程：高级语言—汇编语言—二进制机器指令

软件

软件是计算机系统中程序和文档的总称

  • 文档是为了便于了解程序所需的说明性的资料（如：设计说明书、用户使用手册等）

软件的层次结构（从外到内）：应用软件-系统软件-硬件

应用软件

系统软件

操作系统（operating system 现代计算机必须）

  • 为了使程序更好的在计算机上运行而管理计算机资源的监控程序
  • 用户程序和硬件之间的接口，为用户提供各种服务和监控功能
  • 处理基本的输入和输出操作
  • 分配外存和内存
  • 为多个应用程序提供共享计算机资源和服务

编译程序（compiler 现代计算机必须）

  • 将高级语言翻译为计算机能识别的机器语言的程序
  • 将高级语言编写的程序翻译成硬件能执行的指令

汇编程序

  • 将指令由助记符形式翻译为二进制形式的程序
